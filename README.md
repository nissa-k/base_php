# Exercices PHP — Variables, Conditions, Boucles & Tableaux

Ce dépôt contient une série d’exercices PHP (EX1 → EX19) pour réviser :
- les variables et types
- les conditions (`if / elseif / else`)
- les boucles (`while`, `for`, `foreach`)
- les tableaux (`array`, associatifs, `in_array`, `count`, etc.)

## Prérequis

- PHP 7+ (ou PHP 8+)
- (Optionnel) Un serveur local : WAMP / XAMPP / MAMP

## Comment exécuter

### Option 1 — Ligne de commande
1. Crée un fichier `exercices.php` et colle les exercices dedans.
2. Lance :
```bash
php exercices.php
```
